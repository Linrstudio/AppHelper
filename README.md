## Web App 调试助手

Web App 调试助手 是一个 Chrome 扩展，用于快速查看 Web app 在各种移动终端上尺寸的外观，包含截屏功能。

### 安装

在 https://github.com/Linrstudio/AppHelper/tree/master/download 下载 apphelper.crx 后，拖放到 Chrome 窗口，完成安装。
